# Algorithms (Java)

A curated set of algorithm problems solved in **Java** with:
- Clean, interview-ready implementations
- Unit tests (JUnit 5)
- Time & space complexity notes
- Common patterns cheat sheets

## Structure
- `src/main/java/...` → solutions grouped by topic/pattern
- `src/test/java/...` → tests mirroring the same structure
- `docs/` → patterns + notes
